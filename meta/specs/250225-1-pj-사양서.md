## 개요

`pj`는 `~/projects` 등 지정된 base directory 아래의 프로젝트를 fuzzy finding으로 검색하고, frecency(빈도+최근성) 기반으로 정렬하여, fzf 인터랙티브 UI로 선택 후 이동하는 zsh 함수이다.

**대상 환경**: macOS + zsh 전용

---

## 시스템 요구사항

- macOS
- zsh
- fzf (Homebrew dependency)

## 파일 구조

### 소스 코드

```
pjfzf/
├── pj.zsh              ← 핵심 구현 (단일 파일)
├── Formula/
│   └── pjfzf.rb        ← Homebrew formula
├── tests/
│   └── test_pj.zsh     ← 테스트
└── LICENSE
```

### 런타임 파일

```
~/.config/pj/
├── config              ← base_dirs 설정 (줄 단위)
└── history             ← 접근 히스토리 (pipe-delimited)
```

---

## 설정 파일

**경로**: `~/.config/pj/config`

### 형식

```
# 주석 지원 (# 으로 시작하는 줄)
~/projects
~/work
```

- 한 줄에 하나의 base directory 경로
- `~` 확장 지원
- 빈 줄과 `#` 주석 무시
- 첫 실행 시 `~/.config/pj/` 디렉토리 + config 파일 자동 생성
- 기본값: `~/projects`

---

## 히스토리 파일

**경로**: `~/.config/pj/history`

### 형식

```
디렉토리명|전체경로|마지막접근epoch|접근횟수
```

예시:
```
my-project|/Users/user/projects/my-project|1708851200|5
```

- pipe(`|`) 구분자
- `pj`로 디렉토리 이동 시 자동 업데이트
- 존재하지 않는 경로의 항목은 목록 생성 시 자동 제외

---

## Frecency 알고리즘

### 점수 계산

```
score = access_count × time_weight
```

### time_weight 규칙

| 조건 | 가중치 |
|------|--------|
| 1시간 이내 접근 | ×16 |
| 1일 이내 접근 | ×8 |
| 1주일 이내 접근 | ×4 |
| 1달 이내 접근 | ×2 |
| 그 외 | ×1 |

### 정렬

- 히스토리에 있는 디렉토리: frecency 점수 내림차순
- 히스토리에 없는 디렉토리: 히스토리 항목 뒤에 이름순(알파벳)으로 배치

## CLI 인터페이스

### 서브커맨드

| 커맨드 | 동작 |
|--------|------|
| `pj` | fzf로 프로젝트 선택 후 이동 |
| `pj <query>` | query를 fzf 초기 검색어로 사용 |
| `pj add <path>` | base directory 추가 |
| `pj remove <path>` | base directory 제거 |
| `pj list` | 등록된 base directories 표시 |
| `pj help` | 사용법 표시 |

### 동작 상세

- **`pj` / `pj <query>`**: 프로젝트 목록을 frecency 정렬 → fzf 표시 → 선택 시 cd + 히스토리 업데이트
- **`pj add <path>`**: config 파일에 경로 추가. 중복 시 무시. 경로 존재 확인.
- **`pj remove <path>`**: config 파일에서 경로 제거. 없으면 경고.
- **`pj list`**: config에 등록된 모든 base directory 표시.
- **`pj help`**: 사용법 출력.

## fzf 연동

### fzf 옵션

```
--query="${query}"        # 초기 검색어
--select-1                # 결과 1개면 자동 선택
--exit-0                  # 결과 0개면 바로 종료
--preview 'ls -la {}'     # 미리보기
--preview-window=right:40%
--height=40%
--reverse                 # 역순 레이아웃 (상단에서 시작)
--prompt='pj> '
```

### 표시 형식

fzf에는 전체 경로를 전달하여 선택 시 바로 cd에 사용한다.

## Tab 완성

### `_pj` 함수

- `pj <Tab>`: fzf 인터랙티브 목록 표시
- `pj add <Tab>`: 디렉토리 완성
- `pj remove <Tab>`: 디렉토리 완성
- `pj help <Tab>`, `pj list <Tab>`: 인수 불필요, 완성 없음

`compdef _pj pj`로 등록.
